Email Scraping / Bulk Email GUI

I made a system for scraping the internet for email addresses; I wrote a Python script using selenium (browser automation module) to open up a duckduckgo search page and run search queries. The URL’s from the queries are stored in a list and then traversed to find emails. These emails are sent to a dedicated IP so that multiple computers can scrape at once.

Once the email addresses are collected; I have another script that can send bulk emails from multiple gmail accounts so as not to exceed the daily allowable sent emails for any one account.

GUI that allows the user to upload an Excel with emails and other information. You can dynamically add elements from the Excel file into the body of the email with {}

This is my most artistic project and has yielded surprising results. I’ve used it to blast email potential clients, find art galleries for my paintings, and other mass outreach. You need to be creative to come up with queries to feed into duckduckgo; I’ve used a list of the 100 most common names + a profession to get more results. I’ve also used databases like Common Crawl which is a massive database containing metadata from websites around the internet for more URLs to scrape.

Nobody else does this in my niche; and I’ve connected with a surprising amount of people — I used Craigslist as an inspiration for this project and tried to encapsulate the excitement anonymous connection on the internet in contrast to the nerfed and highly supervised connections you make on Facebook, Linkedin, and Instagram.
