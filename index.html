I used this API to organize the Dropbox account, I renamed the files in a useful fashion and created a hierarchy of information.

Project 3: Blur Detection

Worked on a data collection project. Used FFMPEG to pull frames from source video without any quality loss. Then used Python OpenCV the % blur of each frame to get an "average blur". Every frame that was below two standard deviations of the average blur was deleted. This standarized the data collection and massively sped the mundane process.
