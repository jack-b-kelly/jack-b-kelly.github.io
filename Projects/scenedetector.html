OpenCV Scene Detector

Objective:

Use Python's OpenCV module to open each individual frame in a video and compare it to the previous frame; quantifying the difference as a percentage. Then, use these percent differences to identify when a cut-scene has occurred.

Video Demo

Functionality:

Each frame in a particular video is looked at as an array of values; representing colors and dimension. These arrays can be subtracted from each other and the differences can be analyzed graphically. When visualized in a line plot, we see that spikes in percentage difference can be attributed to scene changes. We also see that individual sequences cluster together.

This program could be pushed further to identify sequence speeds (i.e. the greater percentage difference in a sequence the more motion or speed) We can analyze percent differences and identify sequences where there is a lot of motion or little motion. Hypothetically, we could have a problem where there's alot of video to analyze and we're only interested in parts where there is more comparitive motion on camera. This program could identify those sequences and save a lot of time and energy.

Each frame (as an array) is subtracted from the preceding frame. On the right, we see only the pixels that are different from the previous frame. Pixels that are the same are displayed as black. The screen flashes on cut-scenes because all the pixels are different, and there are more pixels displayed when there is a lot of motion in the sequence.
